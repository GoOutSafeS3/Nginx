# GoOutSafe - ApiGateway

## Running the code

### Unit tests

```
$ ./run.sh unittests
```

### Integration tests

```
$ ./run.sh integration-tests
```

### Frontend tests

```
$ ./run.sh frontend-tests
```

### Deploy with fake data

```
$ ./run.sh deploy-test
```

### Deploy in production

```
$ ./run.sh deploy-prod
```
